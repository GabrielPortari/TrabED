Os programas estão divididos em pastas de acordo com suas respectivas funções

Os arquivos veiculos.ernv deve ser utilizado como parametro de entrada para todos os programas

Para executar os programas, é necessário passar o arquivo veiculos.ernv como parâmetro na execução

Para as máquinas do Laboratório, caso prefira, é possivel executar pela IDE Dev-C++, basta que o usuário vá em Execute > Parameters... e coloque o parametro veiculos.ernv
